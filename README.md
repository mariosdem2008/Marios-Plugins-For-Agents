# Marios Plugins For Agents

Personal backup of local agent skills.

## Layout

- `skills/codex/` - skills from `C:\Users\mario\.codex\skills`
- `skills/agents/` - skills from `C:\Users\mario\.agents\skills`
- `skills/workspace-creator-canon-social-content/` - project-local skills from `C:\Users\mario\Desktop\Creator Canon Social Content\.skills`

Each skill directory keeps its original `SKILL.md` and any supporting files that were present locally.

## Restore

Copy any skill directory back into the matching local skills root, then restart Codex so the skill list refreshes.
